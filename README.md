## 🚀 Проекты

### [📊 WEB Scraping](./quotes-scraper)
Парсинг сайта с цитатами, обработка данных и сохранение в удобный Excel-файл с несколькими вкладками.

**Технологии:** `requests`, `BeautifulSoup`, `pandas`, `openpyxl`

[→ Перейти в проект](./quotes-scraper)

---

### [💰 API Integration](./crypto-prices)
Получение актуальных данных о криптовалютах через CoinGecko API и формирование красивого отчёта.

**Технологии:** `requests`, `pandas`, `openpyxl`

[→ Перейти в проект](./crypto-prices)

---

### [📈 Excel Automation](./excel-sales-report)
Автоматическое создание профессиональных сводных отчётов по продажам из SQLite базы данных.

**Технологии:** `sqlite3`, `pandas`, `openpyxl`

[→ Перейти в проект](./excel-sales-report)

---

## 🛠 Технологии

- **Python 3.9+**
- `pandas` — обработка данных
- `openpyxl` — работа с Excel
- `requests` + `BeautifulSoup` — веб-скрапинг
- `sqlite3` — работа с базами данных
