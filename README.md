# Python Automation Portfolio

Мои проекты по автоматизации задач на Python.  
Здесь собраны примеры работы с веб-скрапингом, API и Excel-отчётами.

## 📌 Проекты

| Проект | Описание | Технологии |
|--------|----------|------------|
| **[WEB Scraping](./quotes-scraper)** | Парсинг сайта с цитатами и сохранение в Excel | `requests`, `BeautifulSoup`, `pandas`, `openpyxl` |
| **[API Integration](./crypto-prices)** | Получение данных о криптовалютах через CoinGecko API | `requests`, `pandas`, `openpyxl` |
| **[Excel Automation](./excel-sales-report)** | Автоматическое создание сводных отчётов из SQLite базы | `sqlite3`, `pandas`, `openpyxl` |

---

## 🚀 Как запустить

Каждый проект имеет свой `README.txt` с подробной инструкцией.

**Общее:**
- Python 3.9+
- `pip install -r requirements.txt`
