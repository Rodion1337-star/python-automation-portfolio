Crypto Prices Fetcher (CoinGecko)
=================================

Simple Python script that downloads the top cryptocurrencies by market cap
from the free CoinGecko API and saves them to crypto_prices.xlsx.


Requirements
------------
- Python 3.9 or newer
- Internet connection
- No CoinGecko API key needed (public Demo API)


Install
-------
1. Open a terminal in this folder.

2. (Recommended) Create and activate a virtual environment:

   Windows (PowerShell):
       python -m venv .venv
       .\.venv\Scripts\Activate.ps1

   macOS / Linux:
       python3 -m venv .venv
       source .venv/bin/activate

3. Install dependencies:

       pip install -r requirements.txt


Run
---
From this folder:

    python fetch_crypto_prices.py

The script will create (or overwrite) crypto_prices.xlsx in the same folder.


Change how many coins to fetch
------------------------------
Open fetch_crypto_prices.py and edit the TOP_N value near the top:

    TOP_N = 50   # default: top 50 by market cap

Examples:
    TOP_N = 20   # top 20
    TOP_N = 100  # top 100

Then run the script again.


Output columns
--------------
- Name
- Symbol
- Current Price (USD)
- Market Cap
- 24h Price Change (%)
- Total Volume (24h)


Rate limits
-----------
The free CoinGecko API has request limits. For TOP_N up to 250 the script
usually needs only one request. If more pages are needed, it waits briefly
between calls (REQUEST_DELAY_SECONDS). If you see HTTP 429, wait a minute
and try again, or increase REQUEST_DELAY_SECONDS in the script.


Example file
------------
crypto_prices.xlsx is an example export generated by running this script.
Run the script again anytime to refresh prices.


Troubleshooting
---------------
- "ModuleNotFoundError": run  pip install -r requirements.txt
- "HTTP 429": rate limited — wait and retry
- Empty or incomplete file: check your internet connection and try again


Data source
-----------
CoinGecko public API:
https://www.coingecko.com/en/api
https://api.coingecko.com/api/v3/coins/markets
